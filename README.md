# k2s.sh

Landing page and installer endpoint for [kubernetes2simple](https://github.com/dekubeio/kubernetes2simple).

Published at **[k2s.dekube.io](https://k2s.dekube.io/)**.

- `index.html` — landing page
- `get/index.html` — `curl -fsSL k2s.dekube.io/get | bash` installer endpoint

Part of the [dekube](https://dekube.io) project.
